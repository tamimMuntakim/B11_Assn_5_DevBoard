Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in DOM. They are -

  • Using document.getElementById() : Selects only one element matching the ID
  • Using document.getElementsByClassName() : Selects all elements matching the class name
  • Using document.querySelector() : Selects first element matching the given CSS selector
  • Using document.querySelectorAll() : Selects all elements matching the given CSS selector
  • Using document.getElementsByTagName() : Selects all elements matching the tag name

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML() : It reads the target element as it is written in the HTML file, including comments
innerText() : It reads the target element's only visible or painted text in the browser ignoring all HTML tags, but including outlook styling
textContent() : It reads the target element's all texts ignoring all HTML tags and styling, but includes hidden texts

Question-3: What is event delegation in the DOM?

Event delegation in the DOM means adding a single EventListener to the parent instead of adding EventListeners to each child nodes. Using the Event Bubbling technique, parent's EventListener can correctly capture the child who triggered the event.

Question-4: What is event bubbling in the DOM?

Event bubbling is a technique in the DOM where an event propagates from the target element to its parent elements using capture, target and bubble phases.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element we use document.createElement() method. It gives us our desired new HTML element.
To add a element we must create a new element using the method mentioned above. Then we select the parent element (of which the new element will be child of) using any preferred method. Then we use parent.appendChild() method to add the new element. Or we can simply select the parent element and modify it's innerHTML to add new elements.
To remove an element we must select it first using any preferred method. The we call remove() on itself. Or we can also select the parent element and use removeChild() method.
